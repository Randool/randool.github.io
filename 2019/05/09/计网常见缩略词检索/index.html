计网常见缩略词检索

发表于 2019-05-09 更新于 2019-05-10 Valine：
本文字数： 1.5k 阅读时长 ≈ 1 分钟

学习计网一开始最头痛的就是一大堆缩略词了，有必要记录一下！

本页面长期维护

  • A
    • AQM——Active Queue Management——主动队列管理
    • ATM——Asynchronous Transfer Mode——异步传输模式
  • B
    • BGP——Border Gateway Protocol——边界网关协议
  • C
    • C/S，B/S
    • CBR——恒定的可变的
    • CDN——Content Distribution Network——内容分发网络
    • CI——Congestion Indication——拥塞指示
    • CWND——congestion window——拥塞窗口简写
  • D
    • DHT——distributed hash table——分布式散列表
    • DNS——Domain Name System——域名系统
    • DSL——Digital Subscriber Line——数字用户线
  • E
    • ECN——Explicit Congestion Notification——显式拥塞通知
    • EFCI——Explicit Forward Congestion Indication——显示转发拥塞指示
    • ER——Explicit Rate——显示速率
  • I
    • IMAP——Internet Mail Access Protocol——因特网邮件访问协议
    • ISO——International Organization for Standardization——国际标准化组织
    • ISP——Internet Service Provider——网络服务提供商
    • IXP——Internet Exchange Point——因特网交换点
  • M
    • MSS——maximum Segment Size——最大报文段长度
    • MTU——Maximum Transmission Unit——最大传输单元
  • N
    • NI——No Increase——无增长
  • O
    • OSI——Open System Interconnect——开放系统互连
    • OSPF——Open Shortest Path First——开放式最短路径优先
    • OSI——Open System Interconnection——开放式系统互联
  • P
    • POP——Point Of Present——存在点
    • POP3——Post Office Protocol version 3——第三版邮局协议
  • Q
    • QoS——Quality of Service——服务质量
  • R
    • rdt——reliable data transfer protocol——可靠数据传输
    • RFC——Request For Comments——请求评议
    • RTT——Round-Trip Time——往返时间
  • S
    • SDN——Software Defined Network——软件定义网络
    • SMTP——Simple Mail Transfer Protocal——简单邮件传输协议
    • SSL——Secure Socket Layer——安全套接字层
    • SYN——Synchronize Sequence Numbers——同步序列编号
  • T
    • TCP——Transmission Control Protoco——传输控制协议
    • TOS——服务类型
    • TTL——Time To Live——该字段指定IP包被路由器丢弃之前允许通过的最大网段数量
  • U
    • udt——unreliable data transfer protocol——可靠数据传输
    • UDP——User Datagram Protocol——用户数据报协议
    • URL——Uniform Resource Locator——统一资源定位符
  • V
    • VC——virtual circuit——虚电路

俚语

  • fly-by-night——骗人的（俚语）
  • nuts and bolts——具体细节，基本要素
  • one’s man meat is another man’s posion——萝卜青菜各有所爱
  • tit-for-tat——针锋相对
